## 🚀 Getting Started
To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/nopal-fz/Website-Personal.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Website-Personal
   ```
3. Open `index.html` in your browser.
